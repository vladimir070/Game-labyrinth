# Maze Game (Лабиринт)

Простая игра "Лабиринт", написанная на Python с использованием библиотеки Tkinter.

## Особенности

*   **Генерация лабиринта:** Используется алгоритм Recursive Backtracker для создания случайных лабиринтов.
*   **Разные уровни сложности:** Выберите один из трех уровней сложности (Новичок, Средний, Профессионал), чтобы изменить размер лабиринта.
*   **Подсказка (A*):** Используйте алгоритм A* для поиска оптимального пути к выходу из лабиринта.
*   **Таймер:** Отслеживайте время, затраченное на прохождение лабиринта.
*   **Управление:** Перемещайтесь по лабиринту с помощью клавиш со стрелками.
*   **Интуитивно понятный интерфейс:** Простое и понятное управление игрой.

## Как играть

1.  **Установите Python:** Убедитесь, что на вашем компьютере установлен Python 3.
2.  **Загрузите код:** Клонируйте репозиторий или скачайте исходный код.
3.  **Запустите игру:** Откройте терминал или командную строку, перейдите в каталог с файлами игры и выполните команду:

    ```bash
    python ваш_файл_с_игрой.py
    ```

    (Замените `ваш_файл_с_игрой.py` именем файла, содержащего код игры).

4.  **Управление:**
    *   Используйте клавиши со стрелками (Вверх, Вниз, Влево, Вправо) для перемещения игрока по лабиринту.
    *   Выберите пункт "Подсказка" в меню "Помощь", чтобы увидеть решение лабиринта.

## Зависимости

*   Python 3
*   Tkinter (обычно входит в стандартную установку Python)

## Структура кода

*   `ваш_файл_с_игрой.py`: Основной файл, содержащий код игры.
*   `MazeGenerator`: Класс, отвечающий за генерацию лабиринта.
*   `MazeSolver`: Класс, реализующий алгоритм A* для решения лабиринта.
*   `MazeGUI`: Класс, создающий графический интерфейс игры с использованием Tkinter.
